# To-Do List Project

A simple, interactive To-Do List web application built with HTML, CSS, and JavaScript. This app allows users to add, edit, delete, and mark tasks as completed, with real-time counters for completed and uncompleted tasks.

## Features

- **Add Tasks**: Enter a task in the input box and click "Add" to include it in the list.
- **Edit Tasks**: Click the "Edit" button next to a task to modify its text.
- **Delete Tasks**: Click the "Delete" button to remove a task from the list.
- **Mark as Completed**: Check the checkbox to mark a task as completed, which applies a strikethrough style.
- **Task Counters**: Displays the number of completed and uncompleted tasks in real-time.

## Technologies Used

- **HTML**: Structure of the web page.
- **CSS**: Styling for the user interface, including a gradient background and responsive design.
- **JavaScript**: Functionality for task management, event handling, and dynamic updates.

## How to Use

1. Open `index.html` in your web browser.
2. Enter a task in the input box.
3. Click the "Add" button to add the task to the list.
4. Use the "Edit" button to modify a task.
5. Use the "Delete" button to remove a task.
6. Check the checkbox to mark a task as completed.
7. View the counters at the bottom for completed and uncompleted tasks.

## File Structure

- `index.html`: The main HTML file containing the structure of the To-Do List.
- `style.css`: The CSS file for styling the application.
- `script.js`: The JavaScript file handling the app's functionality.

## Browser Compatibility

This application works in all modern web browsers that support HTML5, CSS3, and ES6 JavaScript.
